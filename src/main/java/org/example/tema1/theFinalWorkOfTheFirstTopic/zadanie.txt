Анализатор курса валют.
Пользователь вводит дату.
1. Произвести анализ курса валют на эту дату, предыдущую и следующую:
- Вывести, все три  курса
- насколько курс вырос/упал.
- Наибольший и наименьшие значения из этих трех
- Сохранить в отдельную директорию лучший снимок NASA за эту дату :)

2. Продвинутое, усложненное задание:
- Пользователь вводит месяц и год. Вывести курс рубля за этот месяц, найти наибольший и наименьшие значения

3. Очень сложное:
- Найти самые сильные скачки в этот промежуток, дни, когда курс сильно вырос или упал. Автоматически скачать текст статьи из википедии, отвечающей за факты на эту дату (пример https://ru.wikipedia.org/wiki/%D0%A4%D0%B5%D0%B2%D1%80%D0%B0%D0%BB%D1%8C_2021_%D0%B3%D0%BE%D0%B4%D0%B0), и показать пользователю
Для этих заданий можно использовать проекты с пройденных занятий, также можно использовать данные библиотеки:
java.io.BufferedReader;
java.io.IOException;
java.io.InputStream;
java.io.InputStreamReader;
java.net.URL;
java.net.URLConnection;

Критерии оценивания решения 1-ого кейса:
1 балл - представлен анализ валют из домашнего задания
2 балла - в 1 задании реализованы только вывод всех трех курсов
3 балла - выполнено полностью первое задание
4 балла - выполнено полностью 2-е задание
5 баллов - выполнено полностью 3-е задание
